Лабораторна робота №4

На індекс LR3 LR5

1. Тема та мета

Тема: Адаптивна верстка. Медіа-запити. Мета-тег viewport.

Мета: навчитися робити макет сторінки, що коректно відображається на різних пристроях (ПК, планшет, смартфон), з використанням медіа-запитів.

2. Мета-тег viewport

<meta name="viewport" content="width=device-width, initial-scale=1">

Цей тег повідомляє браузеру, що ширина віртуального вікна повинна дорівнювати ширині пристрою. Без нього сторінка на мобільних може виглядати дуже дрібною.

3. Приклад медіа-запиту

/* стилі за замовчуванням (для десктопів) */
.content-wrapper {
  display: flex;
  gap: 16px;
}

/* адаптація для екранів менше 768px */
@media (max-width: 768px) {
  .content-wrapper {
    flex-direction: column;
  }

  .product-card {
    flex: 1 1 100%;
  }
}

4. Типові точки перелому (breakpoints)

  • max-width: 1024px — планшети в горизонтальному положенні.
  • max-width: 768px — планшети / великі телефони.
  • max-width: 480px — смартфони.

5. Приклад адаптивного меню

.nav-flex {
  display: flex;
  gap: 12px;
}

/* для мобільних — меню в колонку */
@media (max-width: 600px) {
  .nav-flex {
    flex-direction: column;
    align-items: flex-start;
  }
}
[Скріншоти: вигляд сторінки на десктопі, планшеті, смартфоні]

6. Висновки

У ЛР4 було налаштовано мета-тег viewport, додано кілька медіа-запитів для зміни розташування блоків на малих екранах та перевірено відображення макета інтернет-магазину на різних пристроях.
