1. Тема та мета

Тема: CSS. Класифікація селекторів. Стильове оформлення елементів HTML-документа.

Мета: вивчити основні типи селекторів CSS та навчитися використовувати їх для оформлення тексту, посилань, списків, таблиць і блоків.

2. Способи підключення стилів

  • Inline-стилі: атрибут style в середині тега.
  • Внутрішні стилі: тег <style> у блоці <head>.
  • Зовнішній файл: <link rel="stylesheet" href="style.css">.

3. Класифікація селекторів

  1. Селектори за тегом: p { color: #333; }
  2. Селектори за класом: .btn { padding: 8px 16px; }
  3. Селектори за id: #header { background: #005bbb; }
  4. Групові селектори: h1, h2, h3 { font-family: Arial; }
  5. Комбіновані селектори: nav a { ... }, .card > h3 { ... }
  6. Псевдокласи: a:hover, input:focus, li:first-child.
  7. Псевдоелементи: ::before, ::after, ::first-line.

5. Стилізація таблиці та списків

  width: 100%;
  border-collapse: collapse;
}
.product-table th,

.product-table thead {
  background: #f0f0f0;

.feature-list {
