1. Тема та мета

Тема: Верстка HTML-документа. Верстка засобами CSS та Flexbox.

Мета: освоїти модуль Flexbox та застосувати його для побудови макета сторінки інтернет-магазину.

2. Основні поняття Flexbox

  • flex-container — елемент з display: flex;
  • flex-items — дочірні елементи контейнера.
  • flex-direction — напрямок розташування елементів (row, column).
  • justify-content — вирівнювання по головній осі.
  • align-items — вирівнювання по поперечній осі.
  • flex-wrap — перенесення елементів на новий рядок.

3. Простий приклад Flexbox

<div class="flex-row">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>

.flex-row {
  display: flex;
  gap: 10px;
}

.box {
  flex: 1;
  background: #e9f2ff;
  border: 1px solid #005bbb;
  padding: 10px;
  text-align: center;
}
[Скріншот: три блоки, вирівняні в один ряд за допомогою Flexbox]

4. Шапка сайту на Flexbox

<header class="header-flex">
  <div class="logo">Computer Store</div>
  <nav class="nav-flex">
    <a href="#">Головна</a>
    <a href="#">Каталог</a>
    <a href="#">Контакти</a>
  </nav>
</header>

.header-flex {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background: #005bbb;
  color: #fff;
  padding: 10px 16px;
}

.nav-flex a {
  margin-left: 12px;
  color: #fff;
  text-decoration: none;
}

5. Макет сторінки (контент + сайдбар)

<div class="content-wrapper">
  <main class="main-content">
    <h2>Каталог товарів</h2>
    <div class="products-grid">
      <article class="product-card">Товар 1</article>
      <article class="product-card">Товар 2</article>
      <article class="product-card">Товар 3</article>
    </div>
  </main>

  <aside class="sidebar">
    <h3>Фільтри</h3>
    <ul>
      <li>По ціні</li>
      <li>По бренду</li>
      <li>По типу товару</li>
    </ul>
  </aside>
</div>

.content-wrapper {
  display: flex;
  gap: 16px;
  margin: 16px 0;
}

.main-content { flex: 3; }

.sidebar {
  flex: 1;
  background: #f0f0f0;
  padding: 10px;
}

.products-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 12px;
  margin-top: 10px;
}

.product-card {
  flex: 1 1 calc(33.333% - 12px);
  background: #ffffff;
  border: 1px solid #ddd;
  padding: 10px;
}
[Скріншот: сторінка з контентом і сайдбаром на Flexbox]

6. Висновки

У ЛР3 було використано Flexbox для побудови макета сторінки: шапки, блоку контенту, сітки товарів та бічної панелі. Flexbox спрощує адаптивну верстку та дозволяє гнучко розташовувати елементи.
